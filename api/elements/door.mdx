## BaseDoor

The `BaseDoor` class represents a door element in OpenPlans. It extends the `Opening` class and provides functionality for creating customizable doors with frames, panels, and rotation controls.

### Constructor

```typescript
new BaseDoor(baseDoorConfig?: OPDoor)
```

<ParamField path="baseDoorConfig" type="OPDoor" optional>
  Optional configuration object for the door. If not provided, default values will be used.
</ParamField>

### Properties

<ParamField path="ogType" type="string">
  The element type identifier. Always set to `ElementType.DOOR`.
</ParamField>

<ParamField path="propertySet" type="OPDoor">
  The current configuration of the door including dimensions, colors, and other properties.
</ParamField>

<ParamField path="subElements" type="Map<SubElementType, THREE.Object3D>">
  Map containing sub-elements of the door (frame, panel, opening, finish).
</ParamField>

<ParamField path="selected" type="boolean">
  Indicates whether the door is currently selected.
</ParamField>

<ParamField path="edit" type="boolean">
  Indicates whether the door is in edit mode.
</ParamField>

### Getters & Setters

<ParamField path="labelName" type="string">
  The display name of the door element.
</ParamField>

<ParamField path="doorPosition" type="[number, number, number]">
  The 3D position of the door in the scene as [x, y, z].
</ParamField>

<ParamField path="doorLength" type="number">
  The width of the door opening.
</ParamField>

<ParamField path="doorThickness" type="number">
  The thickness of the door panel.
</ParamField>

<ParamField path="doorHeight" type="number">
  The height of the door panel.
</ParamField>

<ParamField path="doorRotation" type="number">
  The rotation value for the door panel (between 1 and 2, representing the opening angle).
</ParamField>

<ParamField path="doorQuadrant" type="number">
  The quadrant in which the door opens (1-4).
</ParamField>

### Methods

#### setOPConfig

```typescript
setOPConfig(config: OPDoor): void
```

Updates the door configuration and rebuilds the geometry.

<ParamField path="config" type="OPDoor">
  New configuration object for the door.
</ParamField>

#### getOPConfig

```typescript
getOPConfig(): OPDoor
```

Returns the current door configuration.

**Returns:** `OPDoor` - The current door configuration object.

#### setOPGeometry

```typescript
setOPGeometry(): void
```

Rebuilds the door geometry based on the current configuration. This creates the frame and door panel.

#### showProfileView

```typescript
showProfileView(status: boolean): void
```

Toggles the profile view mode for technical drawings.

<ParamField path="status" type="boolean">
  True to show profile view (outline only), false to show normal view.
</ParamField>

## OPDoor

The configuration interface for door elements.

### Properties

<ParamField path="ogid" type="string" optional>
  Unique identifier for the door element.
</ParamField>

<ParamField path="labelName" type="string" required>
  Display name for the door.
</ParamField>

<ParamField path="type" type="ElementType.DOOR" required>
  Element type identifier.
</ParamField>

<ParamField path="dimensions" type="object" required>
  Dimensions configuration for the door.
  <Expandable title="properties">
    <ParamField path="start" type="{ x: number, y: number, z: number }">
      Start point of the door opening.
    </ParamField>
    <ParamField path="end" type="{ x: number, y: number, z: number }">
      End point of the door opening.
    </ParamField>
    <ParamField path="length" type="number">
      Length (width) of the door opening.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField path="doorPosition" type="[number, number, number]" required>
  3D position of the door as [x, y, z].
</ParamField>

<ParamField path="doorType" type="DoorType" required>
  Type of door (WOOD, GLASS, DOUBLEDOOR, SLIDING, FOLDING, DOUBLEACTION, OTHER).
</ParamField>

<ParamField path="doorHeight" type="number" required>
  Height of the door panel in meters.
</ParamField>

<ParamField path="doorThickness" type="number" required>
  Thickness of the door panel in meters.
</ParamField>

<ParamField path="frameThickness" type="number" required>
  Thickness of the door frame in meters.
</ParamField>

<ParamField path="frameColor" type="number" required>
  Hexadecimal color value for the door frame (e.g., 0x000000).
</ParamField>

<ParamField path="doorColor" type="number" required>
  Hexadecimal color value for the door panel (e.g., 0x8B4513).
</ParamField>

<ParamField path="doorRotation" type="number" required>
  Rotation value for the door opening (1-2, where 1 is fully open, 2 is closed).
</ParamField>

<ParamField path="doorQuadrant" type="number" required>
  Quadrant in which the door opens (1-4).
</ParamField>

<ParamField path="coordinates" type="Array<[number, number, number]>" required>
  Array of 3D coordinate points defining the door geometry.
</ParamField>

## DoorType

Enum defining available door types:

- `WOOD` - Wooden door
- `GLASS` - Glass door
- `DOUBLEDOOR` - Double door
- `SLIDING` - Sliding door
- `FOLDING` - Folding door
- `DOUBLEACTION` - Double action door
- `OTHER` - Other door types

## Example

```typescript
import { BaseDoor, DoorType, ElementType } from 'openplans';

// Create a door with default configuration
const door = new BaseDoor();

// Create a door with custom configuration
const customDoor = new BaseDoor({
  labelName: 'Main Entrance',
  type: ElementType.DOOR,
  dimensions: {
    start: { x: -0.5, y: 0, z: 0 },
    end: { x: 0.5, y: 0, z: 0 },
    length: 1,
  },
  doorPosition: [5, 0, 0],
  doorType: DoorType.WOOD,
  doorHeight: 2.1,
  doorThickness: 0.05,
  frameThickness: 0.15,
  doorColor: 0x8B4513,
  frameColor: 0x000000,
  doorRotation: 1.5,
  doorQuadrant: 1,
  coordinates: [],
});

// Update door properties
door.doorHeight = 2.4;
door.doorColor = 0x654321;
door.doorQuadrant = 2;

// Get current configuration
const config = door.getOPConfig();

// Show profile view for technical drawings
door.showProfileView(true);
```
